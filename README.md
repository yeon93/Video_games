# EDA_video_games
-----

## Summary
  + 1980년~2020년까지 출시된 게임의 지역별, 연도별 판매량 데이터를 통해  
게임 시장의 트렌드 변화와, 앞으로 게임시장은 어떻게 될 것인지, 최종적으로 새롭게 출시할 게임을 설계하고 추천


## Inspiration
  + 지역에 따라 선호하는 게임 장르가 있는가?
  + 지역에 따라 선호하는 플랫폼이 무엇인가?
  + 연도별 게임 장르의 트렌드가 있는가?
  + 연도별 플랫폼의 트렌드가 있는가?
  + 연도별 제작사의 트렌드가 있는가?
  + 출고량이 높은 게임에는 어떤 특징이 있는가?
